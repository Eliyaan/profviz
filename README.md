# Profviz

How to use it: 

First profile your code with `v -prof prof_output1 run .`

Then run profviz:  `profviz prof_output1` 

You can open multiple files `profviz prof_output1 prof_output2 prof_output3`


You can click on the labels of the columns to sort them.
